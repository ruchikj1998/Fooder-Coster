# Parcel (Bundler)
- Dev Build
- Local Server
- HMR - Hot Module Replacement
- File watching algorithm - written in c++
- Caching - Faster Builds
- Image Optimization
- Minification
- Bundling
- Compress
- Consistent Hashing
- Code Spliting
- Differential Bundling - support older browsers
- Diagnostics
- Error handling
- Https support
- Tree shaking - will remove unused code
